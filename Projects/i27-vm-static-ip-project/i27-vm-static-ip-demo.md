## **Project Description**

### 🔧 Requirements:

* Create a **custom VPC**
* Create **two subnets**:

  * One in `us-central1`
  * One in `asia-southeast1`
* Launch **one virtual machine**:

  * Attach **startup script**
  * Assign a **static external IP**
  * Deploy in `us-central1-a`
